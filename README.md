# SQL Database Management System Project

This project demonstrates SQL Server database design and stored procedure implementation.

## Database Tables
- tblQualifications
- tblSpecializations
- tblStates
- tblCities

## Features
- CRUD operations using Stored Procedures
- Soft Delete functionality
- Foreign Key relationships
- Audit columns for tracking record activity
- Data retrieval using JOIN queries

## Technologies Used
- SQL Server
- Stored Procedures
- Relational Database Design
